01. Sobre Mí

¡Hola! Soy Jose Maria Martinez Espino, un apasionado desarrollador Full Stack con más de 3 años de experiencia creando sistemas empresariales que realmente marcan la diferencia.

Mi especialidad es transformar procesos complejos en soluciones digitales intuitivas. He desarrollado sistemas completos para empresas como Hulux, logrando reducir tiempos de gestión hasta en un 60% mediante automatización inteligente.

Más allá del código, me encanta resolver problemas reales. Cada proyecto es una oportunidad para aprender nuevas tecnologías y crear algo impactante. Cuando no estoy programando, disfruto explorando nuevas tendencias tech y contribuyendo a la comunidad de desarrolladores.

Aquí hay algunas tecnologías con las que he estado trabajando recientemente:

  • React 18+
  • Firebase
  • JavaScript ES6+
  • Bootstrap 5
  • Firestore
  • Cloud Functions
